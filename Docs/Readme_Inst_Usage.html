This specifies instructions on how to use xdata for an instructor.

  1. Connect to XData from your course management system; if you are not using a course management system, and using XData directly, you can or login directly open http://*root-url*/XDataWeb, and login with the instructor username and password.
  2. The "Upload Schema" feature allows you to specify a SQL script containing the schema. Click on Choose File, browse to the file and click on Upload button.
  3. The "Upload Sample Data" feature allows you to specify a SQL script containing the initial test data. Choose an appropriate Schema Name from the drop down list, click on Choose File and browse to it, and finally click on the Upload button to submit it.
  4. The "New Database Connection" feature allows you to create a connection to any database for executing student queries. The default database used is xdata.
  5. The "New Assignment" feature allows you to setup a new assignment for the students. You can supply the name and description of the assignment, alongwith the schema name. You can also setup the start and end time for the assignment for imposing a deadline. After supplying all settings click on Submit button.
  6. The "View All Assignments" feature allows you to view all the assignments created by you. You can select a desired assignment and use the features Edit Assignment, Edit Questions, Grade Questions and View Results.
  7. The "View Present Assignment" feature displays the currently active assignments for easy access. You have the same functionality here as "View All Assignments" feature.
