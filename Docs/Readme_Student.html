This specifies instructions on how to use xdata for a student. It uses LDAP ID and password for student authentication but can be extended to any authentication method.

  1. Open http://*root-url*/XDataWeb2013
  2. Login with the student username and password.
  3. The "View All Assignments" feature allows you to view all the assignments given to you. You can select a desired assignment and view its details.
  4. The "View Present Assignment" feature displays the currently active assignments for easy access. You have the same functionality here as "View All Assignments" feature.
  5. The "All Grades" feature allows you to view your grades for all the assignments.
  6. The "Grades of Present Assignment" feature displays the grades for currently active assignments for easy access. You have the same functionality here as "All Grades" feature.
