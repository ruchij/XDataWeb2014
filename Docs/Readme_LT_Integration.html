The admin mode allows for setting up the tool with a course management system.

The below instructions are specific to setting up the tool in moodle system. The options specified in the instructions can be extended to other course management systems as well.

  1. Consumer Key and Shared Secret key should be setup for the purpose of OAuth authorization. The same values should be updated in the tool.jsp file as well.
  2. Add the XData as an external tool.
  3. Give the launch URL as http://*root-url*/tool.jsp. For example: http://www.iitb.ac.in/xdata/tool.jsp.
